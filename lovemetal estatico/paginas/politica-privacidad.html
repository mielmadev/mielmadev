Política de Privacidad

Política de Privacidad

1. Responsable del tratamiento

El responsable de los datos personales recogidos en este sitio es Love Metal. Contacto: lovemetal@gmail.com.

2. Datos recogidos y finalidad

Solo se recogen datos personales que el usuario envía voluntariamente a través del formulario de contacto. La finalidad es responder a consultas o solicitudes.

3. Conservación de datos

Los datos se conservarán únicamente el tiempo necesario para gestionar la consulta o solicitud.

4. Derechos del usuario

El usuario puede ejercer sus derechos de acceso, rectificación, supresión y oposición enviando un correo a lovemetal@gmail.com.

5. Destinatarios

No se cederán datos a terceros, salvo obligación legal.

6. Seguridad

Love Metal adopta medidas técnicas y organizativas adecuadas para garantizar la seguridad de los datos personales y protegerlos contra el acceso no autorizado, pérdida, destrucción o alteración.

Entre las medidas implementadas se incluyen el uso de protocolos seguros de comunicación (como HTTPS), sistemas de control de acceso, y la encriptación de datos sensibles.

Además, se realizan auditorías periódicas para garantizar el cumplimiento de las normativas de protección de datos y la mejora continua de las medidas de seguridad.

7. Uso de cookies

Este sitio web utiliza cookies para mejorar la experiencia del usuario. Las cookies son pequeños archivos de texto que se almacenan en el dispositivo del usuario y permiten recordar sus preferencias y realizar análisis estadísticos. El usuario puede gestionar o desactivar las cookies a través de la configuración de su navegador.

8. Transferencias internacionales de datos

En caso de que los datos personales sean transferidos fuera del Espacio Económico Europeo, Love Metal garantiza que dichas transferencias se realizarán cumpliendo con las garantías adecuadas, como cláusulas contractuales tipo aprobadas por la Comisión Europea.
