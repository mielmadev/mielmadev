Proyectos realizados

Estos son algunos de mis trabajos recientes en desarrollo web y soluciones digitales. Cada proyecto está orientado a mejorar la presencia online, la experiencia de usuario y los resultados de negocio de mis clientes.

🥝 Frutería Mejía — Web y Community Manager

Se desarrolló una página web moderna y fácil de usar para Frutería Mejía, pensada para que cualquier persona pueda encontrar información sobre el negocio, ver productos, ofertas y contactar de forma sencilla. Gracias a esta web y a la gestión activa de sus redes sociales (Instagram, Facebook y canal de WhatsApp), la frutería ha conseguido atraer nuevos clientes, mejorar su imagen y estar más cerca de la comunidad, facilitando que cualquier persona pueda consultar novedades y horarios desde su móvil o computadora. Además, me encargo de publicar contenido, responder dudas y ayudar a que más personas conozcan la frutería, potenciando así su presencia digital.

Desde el punto de vista técnico, el proyecto está construido con HTML, CSS, JavaScript, PHP y MySQL. El diseño es completamente responsivo y accesible (cumpliendo WCAG 2.2 AA), el backend en PHP se conecta a una base de datos MySQL para la gestión de productos y contenidos, y se han aplicado buenas prácticas de SEO y optimización de rendimiento. La web integra enlaces directos a redes sociales y canal de WhatsApp, y dispone de un panel de administración sencillo para que el cliente pueda actualizar la información fácilmente. El enfoque del desarrollo permite la autogestión y está preparado para futuras ampliaciones, como la integración de una tienda online o un catálogo dinámico.

🌐 KOBA - Web interactiva

KOBA es una web interactiva pensada para ofrecer una experiencia dinámica y moderna a cualquier usuario. El objetivo principal fue crear un sitio donde la navegación fuera sencilla y atractiva, permitiendo a las personas interactuar con los contenidos de forma intuitiva y visual, sin necesidad de conocimientos técnicos. Esto ayuda a que cualquier visitante pueda descubrir información y funcionalidades de manera cómoda, mejorando la percepción y la presencia online del proyecto.

Desde el punto de vista del desarrollo, KOBA está construido con React, JavaScript y CSS. Se ha trabajado en una arquitectura de componentes reutilizables, gestión eficiente del estado y animaciones fluidas para enriquecer la experiencia de usuario. El diseño es responsive, adaptándose a cualquier dispositivo, y se han aplicado buenas prácticas de accesibilidad y SEO. El código está preparado para futuras ampliaciones y fácil mantenimiento.

📄 CV Demo (html, css, js)

CV Demo es una herramienta pensada para que cualquier persona pueda mostrar sus habilidades y experiencia profesional de forma visual, moderna y accesible. Permite destacar el perfil profesional de manera clara y atractiva, facilitando que quien lo vea entienda rápidamente la trayectoria y los puntos fuertes del candidato, sin complicaciones técnicas ni barreras de uso.

Desde el punto de vista técnico, el proyecto está desarrollado con HTML, CSS y JavaScript. El diseño es responsive y accesible, adaptado para verse correctamente en cualquier dispositivo. Se han implementado animaciones suaves y una estructura modular para facilitar la personalización y el mantenimiento. El código está optimizado para un rendimiento rápido y cumple buenas prácticas de SEO y accesibilidad.
